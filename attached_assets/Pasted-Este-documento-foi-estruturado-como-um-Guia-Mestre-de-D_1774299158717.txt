# 🚀 DOCUMENTO MESTRE: ECOSSISTEMA QUANTA FLOW (MFORTE)

## 📖 1. A História e a Visão: O "Porquê"
O **Quanta Flow** nasceu da necessidade de escalar a sabedoria de um líder com **30 anos de experiência** em negócios, transformando-a em autoridade digital onipresente. O objetivo é permitir que **Mauro Triumph** (e futuramente outros lojistas) atue como o **Estrategista-Chefe**, enquanto a IA atua como a **Equipe de Execução de Elite**.

**O Grande Objetivo:** Criar um "Mestre de Obras" digital que pegue uma única centelha criativa e a multiplique em artigos, vídeos, podcasts e anúncios validados cientificamente, gerando **Prosperidade Equitativa** e impacto social.

## 🏗️ 2. Sprints de Desenvolvimento (Roadmap Replit)

Para evitar retrabalho, o desenvolvimento deve seguir estas quatro fases:

* **Sprint 1: DNA & Estrutura (Admin):** Configuração de perfis multi-tenant, identidades visuais e chaves de API.

## 💬 4. O Chat Wizard (A Lógica da Centelha)

A interface de usuário é um chat guiado que elimina a complexidade técnica.

1.  **A Pergunta Única:** *"Mauro, qual a ideia de poder de hoje?"*
2.  **O Enriquecimento Autônomo:** A IA identifica a área (Filosofia, Neurociência, etc.) e as fontes de validação (Espinosa, Huberman, Durkheim).
3.  **Sugestão de Headlines:** O sistema propõe 3 ganchos magnéticos baseados na ideia.
4.  **Aprovação e Produção:** Uma vez que o usuário escolhe a Headline, o sistema gera o "Pacote MFORTE" de conteúdos.

## 📊 7. Lógica de Código: Database & Backend (Replit)

O desenvolvedor deve implementar este Schema para suportar a escala SaaS:

### Schema de Banco de Dados (SQL)
```sql
-- Identidade da Marca
CREATE TABLE BrandProfiles (
    account_id UUID PRIMARY KEY,
    brand_name TEXT, -- ex: Mauro Triumph
    tone_of_voice TEXT, -- ex: Visionário
    leadership_style TEXT, -- ex: Hormozi
    cloning_ids JSON, -- IDs ElevenLabs/HeyGen
    [cite_start]colors JSON -- [cite: 63]
);

-- Log de Inteligência
CREATE TABLE ContentLibrary (
    id SERIAL PRIMARY KEY,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    user_input TEXT, -- A ideia bruta
    category TEXT, -- ex: Epigenética
    validation_sources TEXT, -- ex: Bruce Lipton
    status TEXT -- Idea, Generating, Ready
);

-- Cronograma
CREATE TABLE PublicationSchedule (
    asset_id INT REFERENCES GeneratedAssets(id),
    platform TEXT, -- ex: Instagram, LinkedIn
    scheduled_time TIMESTAMP,
    status TEXT -- Planned, Sent, Manual
);
```

### Lógica do Chat Wizard (Python/FastAPI)
```python
@app.post("/generate-mforte-content")
async def generate_content(user_idea: str, account_id: UUID):
    # 1. Recuperar DNA da Marca no Admin
    brand = db.get_brand(account_id)
    
    # 2. IA Enriquecer a ideia (Pesquisa de Fontes)
    enriched_data = ai.research_sources(user_idea) #
    
    # 3. Gerar Ativos (Multimodal)
    assets = ai.generate_all_formats(enriched_data, brand.style)
    
    # 4. Criar entrada na ContentLibrary e retornar para aprovação
    return db.save_and_display(assets)
```
